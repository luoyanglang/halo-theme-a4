# 安装指南

## 环境要求

- Halo 2.0 或更高版本
- JDK 17 或更高版本

## 安装步骤

### 方法一：通过 Halo 应用市场安装（推荐）

1. 登录 Halo 管理后台
2. 进入「外观」->「主题」
3. 点击「安装」按钮
4. 搜索「A4」主题
5. 点击「安装」并等待安装完成
6. 启用主题

### 方法二：手动安装

1. 下载主题文件
```bash
# 下载最新版本
wget https://github.com/your-repo/halo-theme-a4/releases/latest/download/theme-a4.zip

# 或者克隆仓库
git clone https://github.com/your-repo/halo-theme-a4.git
```

2. 上传主题
- 登录 Halo 管理后台
- 进入「外观」->「主题」
- 点击「安装」->「本地上传」
- 选择下载的 zip 文件或打包的主题文件夹
- 等待上传完成

3. 启用主题
- 在主题列表中找到「A4」主题
- 点击「启用」按钮

## 初始配置

### 1. 基础设置

进入「外观」->「主题」->「A4」->「设置」

**布局配置**
- 首页宽度：选择 small/middle/big
- 首页标题位置：选择 left/center
- 文章页宽度：选择 middle/big
- 文章标题位置：选择 left/center

**头像配置**
- 上传头像图片
- 选择是否使用圆形头像
- 选择是否启用旋转效果

**页脚配置**
- 设置首页页脚文字
- 设置文章页脚文字
- 设置版权信息

### 2. 配色方案

**预设配色**
- 灰白配色（默认）：经典的黑白灰配色
- 绿金配色：护眼的绿色和金色搭配

**自定义配色**
- 选择「自定义」配色方案
- 分别设置 8 个颜色变量
- 实时预览效果

### 3. 工具开关

根据需要开启或关闭以下功能：
- 暗黑模式
- 回到顶部按钮
- 回退页面按钮
- 文章顶部目录
- 文章左侧悬浮目录

### 4. 字体设置

- 选择是否启用字体切换
- 选择使用霞鹜文楷或思源宋体

### 5. 文章设置

配置文章页面显示的信息：
- 更新时间
- 字数统计
- 阅读时间
- 阅读量
- 上下篇导航

### 6. 图片画廊

配置图片查看器功能：
- 缩略图
- 缩放
- 旋转
- 自动播放
- 全屏

## 创建必要页面

### 创建首页

1. 进入「内容」->「页面」
2. 点击「新建」
3. 标题设置为「首页」
4. 别名设置为「index」
5. 编写首页内容（支持 Markdown）
6. 发布页面

### 创建归档页

归档页由主题自动生成，无需手动创建。

### 创建标签页

1. 进入「内容」->「页面」
2. 点击「新建」
3. 标题设置为「标签」
4. 别名设置为「tags」
5. 选择模板「标签页」
6. 发布页面

### 创建分类页

1. 进入「内容」->「页面」
2. 点击「新建」
3. 标题设置为「分类」
4. 别名设置为「categories」
5. 选择模板「分类页」
6. 发布页面

## 配置菜单

1. 进入「外观」->「菜单」
2. 添加菜单项：
   - 首页：/
   - 归档：/archives
   - 标签：/tags
   - 分类：/categories
   - 其他自定义页面

## 验证安装

1. 访问网站首页，检查布局是否正常
2. 访问文章页，检查样式是否正确
3. 测试暗黑模式切换
4. 测试回到顶部等工具按钮
5. 在移动设备上测试响应式布局

## 常见问题

### 样式不正常

1. 清除浏览器缓存
2. 检查主题是否正确启用
3. 检查 Halo 版本是否符合要求

### 首页显示空白

1. 确认已创建首页页面
2. 确认页面别名为「index」
3. 确认页面已发布

### 字体未加载

1. 检查网络连接
2. 检查字体文件是否完整
3. 尝试切换字体设置
